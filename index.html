    bootNet

    bootNet is a wrapper that adds a bootstrap process to the glmNet package. This script is currently set up to analyse methylation data in the form of beta matrices. Hopefully, future updates will allow analysis of various types of data.

    Quick start

    Using bootNet

                                        
# BootNet
bootNet(data = x, outcome = y, Alpha = 0.1, iter = 1000, sub_sample = 0.666, sampleID = sampleID, method = method)

# Parallel
bootNet.parallel(data = x, outcome = y, Alpha = 0.1, iter = 1000, sub_sample = 0.666, cores = 4, sampleID = sampleID)
